## Learning Outcomes: SPA vs MPA

- Understand the basic difference between Single Page Applications (SPA) and Multi Page Applications (MPA).
- Learn how page loading works in SPA (no full page reload) and in MPA (full page reload).
- Identify real-world use cases where SPA is more suitable and where MPA works better.
- Compare performance, user experience, and SEO aspects of SPA and MPA.
- Gain clarity on when to choose SPA or MPA while building web applications.
